Cara pakai Setup Inventory

Gunakan halaman ini untuk tambah, edit, dan nonaktifkan master data. Jangan menghapus baris langsung dari sheet kalau sudah ada transaksi.

1

Pilih tab master

Kategori, Item, Resep Produksi, atau Resep Preparasi.

2

Tambah atau edit data

Gunakan tombol tambah, lalu isi field utama. ID lama tetap boleh dipakai untuk sinkron data lama.

3

Nonaktifkan, bukan hapus fisik

Tombol hapus akan menandai data tidak aktif agar histori jurnal tetap aman.

4

Refresh setelah simpan

Data akan dimuat ulang setelah simpan. Kalau browser bandel, tekan Ctrl + F5.

Catatan dapur

Resep adalah standar. Jumlah fisik nyata tetap bisa disesuaikan saat transaksi Preparasi atau Produksi. Sisa/susut tidak dihitung otomatis di master.
